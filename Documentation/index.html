1. Getting Started

A bit about getting started with your WordPress project.

1.1 Included Files

After downloaded the theme package open Appster Package folder and unzip it if you haven't already. Inside the package you will find the following files:

  • Documentation
  • Extras
    • import
    • layouts
  • Psd
  • Theme
    • Appster
    • appster-theme.zip

1.2 Theme Installion

To install your purchased WordPress theme what you’ve already downloaded and unzipped you have two ways: WordPress Upload or FTP Upload. Learn a bit about each below in Sections 1.2.1 and 1.2.2

1.2.1 Wordpress Upload

Log in to your brand new installed wp and navigate to Dashboard Appearance Themes Add New and choose the Upload tab at the top.

When your theme is successfully installed, click Activate to activate the newly uploaded theme.

When uploading theme with this method, please make sure that you're uploading the appster-theme.zip file, not entire package you downloaded. If you upload the wrong file you will get an error stating “Error: missing style.css stylesheet.”

1.2.2 FTP Upload

To upload your new WordPress theme with the manual method, login to your website and unzip the appster-theme.zip file and upload it's contents to "wordpress/wp-content/themes" folder.

Once uploaded, navigate to Dashboard Appearance Themes in the WordPress Dashboard and hit Activate below your newly uploaded theme.

1.3 Plugin Activation

At the top of every page in the Dashboard, you will see a WordPress notice to activate the new theme's required plugins. Let's activate them by clicking on Begin Install, and when the plugin(s) are successfully installed, select "Return to required plugins installer" and activate them.

If you did everything correctly the notice at the top of every page is disappeared.

1.4 Import Demo Content

If you need your theme to be exactly the same like in the theme demo, you will need to import the demo content and options, it can be good starting when heading to customize your site.

For WordPress newbies it's highly recommended to import the demo content, it makes the customization process easier.

In the following sections 1.4.1 - 1.4.5 you can learn more about importing the demo content if you choose to start your site with a completely working example, the same in the theme demo.

1.4.1 Importing Contents

For first you will have to import the demo xml file what contains the backup of pages, posts and contents of the theme demo. To do that head to Dashboard Tools Import and click on the Wordpress Importer and install it at the upcoming page.

Once it's ready, browse the appster-demo-content.xml file from the "Extras/Import" folder (in the downloaded theme package) and hit Upload file and Import. You will be redirected to a page where the authors of the imported contents are listed, never mind, just make sure to check "Download and import file attachments" and hit Submit to start the process.

1.4.2 Importing Theme Options

To import the same theme options used in the live preview open the appster-theme-options.txt in a notepad and select all of the weird looking characters then copy them and paste in Dashboard Appearance Options Backup to the "Import Theme Options" field.

After simply hit the Import Theme Options button and you're done.

1.4.3 Set Front Page Displays

Go to Dashboard Settings Reading and change the "Front Page Displays" to Static Page and select "Home" for your front page and "Blog" for your posts page and finally hit Save Changes.

1.4.4 Set Permalink Structure

Navigate to Dashboard Settings Permalinks and choose Post Name from the list and hit Save Changes

1.4.5 Set Primary Menu

Head to Dashboard Appearance Menus and select Appster Menu from the dropdown menu selector at the top. Check the "Top Primary Menu" checkbox at the bottom and click Save Menu

If the steps above were done in the right way, your site should look the same like in the theme demo.

2. The Home Page

Setting up and customization of the static one-page home page

2.1 Introduction

The home page contains a dozen of different sections to cover every purpose of usuage of an app landing page. Every section has it's own tab with many options in the Theme Options panel, you can see through them at Dashboard Appearance Theme Options


You will notice that most of the elements has an option of selecting it's animation. You can check the available animations in case you are not familiar with them.

2.2 Creating The Home Page

To set up the home page add a new page by go to Dashboard Pages Add New and after adding your wished page title (e.g: "Home") set the Page Template on the right to "Home", and hit Publish

If you checked your site and don't see the empty home page please look back to Section 1.4.3 and set your Home Page as the site's default front page.

To add a predefined section to your home page (e.g. Team Section), first create a New Section at Dashboard Appearance Theme Options General Home Sections Add New and choose a section type from the dropdown menu and hit Save Changes.

3. The Nav Menu

Short guide about setting up the navigation menu.

3.1 Introduction

The idea behind Appster theme's navigation menus are basically having two types of menu items:

  1. Use Page based items for separate pages like Blog or any other pages what's not on the Home Page.
  2. Create Link based items for sections what are located on the Home Page and just scrolling is needed.

You can read more on how to use the two types in Section 3.3 and Section 3.4

3.2 Creating New Menu

To set up the navigation menu from scratch, head to Appearance Menus and select Create New Menu at the top of the page to create a new menu. Name it whatever you want, but make sure that the checkbox near
Top Primary Menu is checked and finally hit Save Menu

3.3 Page Menu Items

Use Page items on the left when you are creating menu links for separate pages, just select the needed pages from the list and click Add To Menu

3.4 Link Menu Items

Use Link type on the left when you are creating scrolling links for the Home Page sections by adding your site url and the Unique section ID with a # sign before, name it as you wish and finally hit Add To Menu

3.5 Example

For example if you want to add a "Pricing" menu item in the navigation what scrolls down to the pricing section when clicked, do the following:


  1. Check the pricing section's Unique Section ID at Appearance Theme Options Pricing

  2. Go back to the menus and add your basic home url to the URL field

  3. Add a # character and the pricing section's Unique section ID right after it

  4. Name the menu item and hit Add To Menu and finally save the menu.

4. Content Editing

Adding pages and shortcodes supported by the theme

4.1 Creating Pages

To create separate custom pages from the home page and it's sections, navigate to Pages Add New and set the page template at the right to Default and hit Publish

4.2 The Shortcodes

You can use custom shortcodes included with the theme in separate custom pages or every advanced textarea in the Theme Options. At the following sections you can see what shortcodes are supported with this theme.

4.2.1 Columns Shortcode

Appster has 16 columns at full width by default. You can easily add responsive columns to your content, just take a look at the code sample of the usuage.

[column width="1/2"]This is the first half column[/column]
[column width="1/2"]This is the second half column[/column]
Width Options
  1. 1 = full column
  2. 1/2 = half column
  3. 1/3 = one-third column
  4. 2/3 = two-third column
  5. 1/4 = one-fourth column
  6. 3/4 = three-fourth column

4.2.2 Divider Shortcode

Divider shortcode is a 1px full width solid grey divider, it can be used for separating different content sections

[divider]

4.2.3 Titles Shortcode

You can define titles for your content from a variety of h1-h6 tags. You can see the theme demo's Shortcodes section for reference of the headlines layout.

[title type="h2"]This is a h2 headline[/title]
Type Options
  1. h1 = h1 heading
  2. h2 = h2 heading
  3. h3 = h3 heading
  4. h4 = h4 heading
  5. h5 = h5 heading
  6. h6 = h6 heading

4.2.4 Button Shortcode

You can easily add buttons with this shortcode, just set the size, color, and URL for the button.

[button size="large" color="color" url="http://www.link.com"]Click Me[/button]
Size Options
  1. small = small size
  2. medium = the default size
  3. large = enlarged size
Color Options
  1. color = colored button, inherits the main color scheme of the theme
  2. light = light grey color
  3. dark = dark grey color
URL Options
  1. url = full url where the button points

5.3.2 Set Up Contact Form

This theme uses Contact Form 7 plugin for the basic contact form. To create your own follow the steps below:

  1. Make sure Contact Form 7 plugin is installed and activated successfully
  2. Head to Dashboard Contact Add New
  3. Select a language for the contact form and name it as you wish
  4. Browse the contact-form-sample.txt from the "Extras/Layouts" folder in the downloaded package and copy paste the content to the contents of the newly created form in the wordpress admin.
  5. Set your email address at the To: field below the form settings and hit Save
  6. Copy the shortcode of the created contact form and paste it the the "Contact Form 7 Shortcode" field at Dashboard Appearance Theme Options Footer Contact Form

6. Troubleshooting

Solutions for common issues and frequently asked questions

My home sections don't display

  1. Make sure your "Front Page Display" is set to a "Static Page" at Settings Reading
  2. Add a section to your home page at Appearance Theme Options General Home Sections
  3. Make sure your section's display is set to "On" at Appearance Theme Options Team Section for example if you added "Team Section" in step 2.
